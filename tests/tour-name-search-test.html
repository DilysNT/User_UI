🔍 Expected Behavior:

  • Tìm kiếm theo tourName sẽ match với tour.name, tour.location, và tour.destination
  • Tìm kiếm không phân biệt hoa thường (case-insensitive)
  • Khi có tourName, nó sẽ bỏ qua các filter khác (destination, departure, dates)
  • Debug component sẽ hiển thị số lượng tours trước và sau filter
  • Console logs sẽ show chi tiết quá trình filter

🐛 Debugging Steps:

  1. Mở Developer Tools (F12)
  2. Click vào một test link
  3. Kiểm tra Console logs để xem:
    • Total tours fetched từ API
    • tourName parameter được đọc từ URL
    • Số tours sau khi filter
  4. Kiểm tra debug component trên trang (màu xanh)
  5. Verify kết quả hiển thị đúng
